# Java Design Patterns - Practice Repository 🚀

Welcome to my personal workspace for studying software design patterns. This repository contains my code implementations, exercises, and personal notes as I read through **"Java Design Patterns"** by **Amit K**.

## 🎯 Purpose
The goal of this repository is to master object-oriented design principles and write cleaner, more maintainable Java code. 

## 📁 Repository Structure
The project is organized by pattern categories for easy navigation:

*   **📂 Creational Patterns** (e.g., Singleton, Factory, Builder)
*   **📂 Structural Patterns** (e.g., Adapter, Decorator, Facade)
*   **📂 Behavioral Patterns** (e.g., Strategy, Observer, Command)

## 🛠️ Tech Stack
*   **Language:** Java (JDK 17+)
*   **Build Tool:** Maven / Gradle
*   **IDE:** IntelliJ IDEA / Eclipse

## 📝 Disclaimers
This code is written for educational and practice purposes only. All conceptual credit for the examples goes to the author, Amit K.
